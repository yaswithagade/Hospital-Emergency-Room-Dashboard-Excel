# 🏥 Hospital Emergency Room Dashboard (Microsoft Excel)

## 📌 Project Overview

This project is an interactive Hospital Emergency Room Dashboard developed using Microsoft Excel.

The dashboard provides insights into emergency room patient data by tracking patient volume, waiting time, satisfaction score, admissions, department referrals, age distribution, and gender analysis.

This project was completed as part of my Excel and Data Analytics learning journey to strengthen my dashboard development and data visualization skills.

## 🎯 Project Objectives

The objective of this dashboard is to help hospital administrators monitor emergency room performance through interactive visualizations and key performance indicators.

The dashboard enables users to

- Monitor total patient count
- Track average waiting time
- Analyze patient satisfaction
- Compare admitted vs non-admitted patients
- Analyze patients by gender
- View department referrals
- Analyze age group distribution
- Filter data using Month and Year slicers

## Data Cleaning Process

The dataset was cleaned using Excel Power Query before analysis.

The following transformations were performed:

- Removed duplicate records
- Corrected data types
- Removed unnecessary columns
- Standardized text values
- Handled missing values
- Renamed columns
- Prepared the data for Pivot Table analysis
